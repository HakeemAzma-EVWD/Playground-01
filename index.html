File naming

  • Keep file names short.
  • They should be descriptive.
    (e.g. "page2.html" vs "about-us.html")
  • No spaces, use underscores or hyphens instead.
  • Use lower case.
    ("About-Us.html" vs. "about-us.html")

File organisation

  • The main folder is called our root folder.
  • The homepage must be index.html
  • index.html must be in the root folder, and not a sub-folder.
  • Other pages can be in subfolders, depending on how you want to stay organised.
  • Images should be placed in a sub folder for better organisation.
