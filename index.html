<a>

Short for "anchor"

Used to link either to a different location within the current page
or to another page.

But how do we tell it where to link to?

<a>google</a>

The word google here is a link someone can click on...
but how does the browser know where to link to?

ATTRIBUTES!

Elements can have attributes

Attributes are always within the opening tag.

Attributes give extra information to the browser about that element,
such as where a link goes, or the location of an image file.

Attributes are normally followed by an equal sign and quotation marks.

<a href="https://google.com">google</a>

Some element require an attribute

A link will not work without an href attribute.

<a href="https://google.com">google</a>

Absolute paths

If the page you are linking to is NOT part of your website,
it MUST start with http:// or https:// to let the browser know
it is an external site that you are linking to.

<a href="https://google.com">google</a>

Relative paths

You can link to other pages within your site by simply putting their file name.


Visit our <a href="about-us.html">about us</a> page!
Visit our <a href="team/john.html">about us</a> page!
